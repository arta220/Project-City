# Тестовый сценарий для провайдеров графиков

## Описание

Эти тесты демонстрируют работу провайдеров графиков для визуализации статистики производства.

## Что проверяют тесты

### IndustrialProductionGraphProviderTests

1. **CardboardProductionGraphProvider_CreatesPlotModel**
   - Проверяет создание модели графика для производства картона
   - Проверяет наличие осей и серий данных
   - Проверяет корректность заголовка

2. **PackagingProductionGraphProvider_CreatesPlotModel**
   - Проверяет создание модели графика для производства упаковки
   - Проверяет наличие осей и серий данных
   - Проверяет корректность заголовка

3. **GraphProviders_HaveCorrectSystemNames**
   - Проверяет корректность имен систем для идентификации графиков

## Демонстрация работы

Тесты показывают:
1. **Создание графиков** на основе статистики производства
2. **Настройку осей** (X - время, Y - количество)
3. **Добавление серий данных** (производство и использование материалов)
4. **Интеграцию** с IndustrialProductionService

## Использование в приложении

Графики доступны через:
- `GraphService.GetAvailableGraphs()` - получение списка доступных графиков
- `GraphService.GetGraphProvider("Производство картона")` - получение конкретного графика
- Окно графиков в WPF приложении (`ChartsWindow`)
