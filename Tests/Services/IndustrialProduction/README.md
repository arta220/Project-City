# Тестовый сценарий для сервиса производства

## Описание

Эти тесты демонстрируют работу `IndustrialProductionService` - сервиса, который отслеживает статистику производства на заводах.

## Что проверяют тесты

### IndustrialProductionServiceTests

1. **IndustrialProductionService_TracksCardboardProduction**
   - Проверяет, что сервис отслеживает производство картона
   - Проверяет запись истории в статистику

2. **IndustrialProductionService_TracksPackagingProduction**
   - Проверяет, что сервис отслеживает производство упаковки
   - Проверяет запись истории в статистику

3. **IndustrialProductionService_TracksMultipleBuildings**
   - Проверяет, что сервис может отслеживать несколько заводов одновременно
   - Проверяет корректную агрегацию данных

4. **IndustrialProductionService_UpdatesOverTime**
   - Проверяет, что сервис обновляется с течением времени
   - Проверяет накопление истории за несколько тиков

## Демонстрация работы

Тесты показывают:
1. **Регистрация заводов** в BuildingRegistry
2. **Обновление сервиса** при каждом тике симуляции
3. **Сбор статистики** по производству
4. **Накопление истории** для построения графиков

## Связь с графиками

Статистика, собираемая этим сервисом, используется провайдерами графиков:
- `CardboardProductionGraphProvider` - для графика производства картона
- `PackagingProductionGraphProvider` - для графика производства упаковки
